# Pindie
Website with games, using react, next.js, express.js + MongoDB
<h2>Run project</h2>

1. Installing dependencies for Back and Front

```bash
npm i
```
2. Run Backend on localhost:3001

```bash
npm run start
```
3. Run Frontend on localhost:3000

```bash
npm run dev
```
